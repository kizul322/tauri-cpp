
本ソースは記事「TauriにC++を組み込んでGUIアプリケーションを作ろう」の全体ソースになります。

# ビルド方法  

以下のツールをご用意ください  
* pnpm
* cmake
* C++20対応のC/C++コンパイラ  
* Rustツールチェイン

インストール後、以下をターミナルで実行ください。  
```sh
pnpm i

# C/C++サイドをビルド
cd ./src-cpp
cmake -B./build_lib && cd ./build_lib
cmake --build

# Tauri(TS+Rust)サイドをビルド(dev起動の場合)
cd ../../
pnpm run tauri dev
```
